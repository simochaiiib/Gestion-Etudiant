Gestion Etudiant
---------------------------------

Ce projet est une application web développée dans le cadre de notre formation à l’École Marocaine des Sciences de l’Ingénieur (EMSI) – Tanger.
L’objectif est créer une application web simple pour gérer les informations des étudiants.Il permet d’ajouter, supprimer et consulter les données des étudiants afin de faciliter la gestion administrative.


Fonctionnalités principales
----------------------------------
Fonctionnalités principales :
Ajouter un étudiant
Connexion (login) 
Inscription (signup)
Gestion des notes 
Dashboard
Emploi du temps (photo)
